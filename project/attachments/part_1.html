HTML is used for creating the primary content of a webpage, giving it a structure.

  • Hyperlinks - can connect two elements of a page: internal hyperlink (this one goes to HTML-5 elements) or two separate pages: external hyperlink (this one goes to the homepage).

  • Images - can be inserted in a page.

  • Frames - used to display a HTML page within a window in another HTML page.


  • Hotspots/image maps - used to create an image with clickable links.

HTML-5 elements include...

  • Audio - to insert an audio recording in a HTML page.


  • Video - to insert a video recording in a HTML page.


  • SVG (Scalable Vector Graphics) - draw shapes/logos using XML.

  • Canvas API - draw shapes/logos using Javascript and Canvas.


  • Geolocation - returns accurate location data.





CSS enables you to distinguish between presentation and content by modifying the design and display of HTML elements.

CSS-2 concepts

The box model

  • Margin - the space between element and others surrounding it.
    Padding - the space between element and its own border.

  • Padding/margin can be inline (horizontal spacing) or block (vertical spacing).

  • Border - the line of separation between the element and others surrounding it.

Types of CSS (where it can be placed)

  • Inline - located in the tag of the element

  • Internal - located in the style div in the document head

  • External - located in an external stylesheet (linked in document head)

CSS-3 concepts
